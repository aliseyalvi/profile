Bizara

Project Brief:

Bizara is cross plateform mobile app developed using React-Naive, Mobx State Tree and Firebase. This is an appointment booking mobile application which connects customers and business owner. Business Owner can create his business in a specific location and set his business hours and customer can search for available businesses list in his location, view available slots and book an appointment.

E-Qari

Project Brief:

E-Qari is cross plateform mobile app developed using React-Naive and Redux. E-Qari is a Quran learning application for muslims. It helps users to learn quranic recitation by actively listening recitation of user and highlighting mistakes and telling accuracy of recitation in realtime. Alquran.cloud open source API is used for quranic arabic, translation and audio data. Our team has developed an Arabic ASR (Automatic Speech Recognition) system using Kaldi Audio Toolkit. This ASR system is hosted on digitalocean droplet and E-Qari application communicate with server through websockets.

Project info

  • Date - 2021
  • Final Year Poject
  • Tools - React Native, Redux, Alquran.cloud API, digitalocean
